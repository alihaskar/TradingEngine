About Trading Engine

Trading Engine also known as TE, is a pragmatic implementation of a marketplace. The TE is a market place solution that comes with instrument model, price / time matching and market data distribution.

The trading engine provides a REST / WEB Socket API allowing clients to connect to the trading engine.

The solution is provided as an open source code prodject under the Apache 2.0 licensing. Currently a version 1.1 is released and further work is in progress. For future works see future plans

For Whom
For any one that like to setup and run a market place or further develop the solution to meet own requirements.

Deploying Trading Engine

  • TE is provided as a public GIT HUB project.
  • The project contains pre-configurations allowing the market place to run out of the box.
  • There are scriptd for starting the components, adding users, and creating trade reports.
  • The TE has been developed with simplicity and performance in mind. Typically requests executes in sub milliseconds. If you are interested in the perfomance characteristics you will find futher information here.
  • A more comprehensive description of the environment, setup and pre-configuration is found here
